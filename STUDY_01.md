## 🧠 핵심 가치

### 1. 속도 — Speed over Perfection
완벽한 설계보다 빠른 실행이 먼저입니다.
**"일단 만들어보자!"**

### 2. 대화 중심 개발 — Conversational Development
코드를 직접 쓰는 대신, **설명 → 생성 → 수정** 루프를 반복합니다.

**예시 대화 흐름**
```
"로그인 기능 만들어줘"
  → "에러 처리 추가해줘"
    → "React로 바꿔줘"
```

### 3. 점진적 구체화 — Iterative Refinement
처음부터 완벽할 필요 없습니다.
계속 질문하고 대화하면서 결과물을 개선해 나갑니다.

### 4. 맥락 위임 — Delegation to AI
**How(어떻게 구현할까)** 는 AI에게 맡기고,
사람은 **What(무엇을 만들까)** 과 **Why(왜 필요한가)** 에 집중합니다.

### 5. 직관 기반 흐름 — Flow-driven
먼저 화면을 띄운 뒤 즉흥적으로 반응합니다.
> "어? 저 버튼이 너무 작네? 키워줘"

## 🛠️ 바이브 코딩에는 어떤 것을 사용할까? (AI 코딩 도구 소개)

### 🎨 Group 1. 초심자 친화형: "말하는 대로 그려지는 빌더"

코드를 전혀 몰라도 괜찮아요! 아이디어를 말로 설명하면 바로 결과물이 만들어지는 도구들이에요.

- **Lovable (러버블):** "쇼핑몰 만들어줘", "포트폴리오 사이트 만들어줘"처럼 채팅으로 요청하면 실제로 동작하는 완성된 웹사이트를 뚝딱 만들어주는 도구예요. 코딩 경험이 전혀 없어도 바로 쓸 수 있어요.
- **v0 (v0.dev):** 버튼, 입력창, 카드 같은 화면 구성 요소(UI)를 채팅으로 설명하면 눈 앞에 바로 그려주는 도구예요. "파란색 로그인 화면 만들어줘"처럼 요청하면 완성된 디자인과 코드를 함께 만들어줘요.
- **Replit (리플릿):** 내 컴퓨터에 아무것도 설치하지 않고, 인터넷 브라우저만으로 코딩부터 실제 배포(남들이 볼 수 있게 공개)까지 한 번에 할 수 있는 도구예요. 입문자에게 특히 추천해요.

### 💻 Group 2. 전문가형 에이전트: "코드와 소통하는 에디터"

이미 어느 정도 코딩을 하고 있는 분들을 위한 도구예요. AI가 내 프로젝트 전체를 이해하고 함께 개발을 도와줘요.

- **Cursor AI (커서):** VS Code를 기반으로 제작된 AI 전용 편집기입니다. 내 프로젝트 전체 코드를 깊이 있게 학습하여 "이 함수 고쳐줘", "버그 찾아줘"와 같은 요청을 가장 정확하게 처리하며, 현재 AI 기반 IDE 중 가장 높은 완성도를 자랑합니다.
- **ChatGPT Codex (코덱스):** OpenAI의 모델을 기반으로 하며, 다양한 개발 환경(IDE)에 플러그인 형태로 연결하여 사용합니다. 코드 생성, 설명, 최적화 등 범용적인 AI 코딩 지원에 강점이 있습니다.
- **GitHub Copilot (깃허브 코파일럿):** 전 세계 개발자들의 코드 저장소인 GitHub에서 만든 AI 코딩 보조 도구입니다. 방대한 오픈 소스 데이터를 학습하여 실시간 코드 추천은 물론, 스스로 계획을 세우고 버그 수정 및 테스트까지 수행하는 '에이전트' 기능이 강력합니다.
- **Google Antigravity (안티그래비티):** 구글이 2025년 11월에 출시한 최신 AI 개발 도구입니다. VS Code 확장 프로그램 형태로 동작하며, AI 에이전트가 코드 작성부터 브라우저를 활용한 실제 작동 테스트까지 자율적으로 처리해 주는 것이 특징입니다.
- **Claude Code (클로드 코드):** 최근 가장 주목받는 '바이브 코딩' 도구로, 터미널(명령창) 환경에서 AI와 직접 대화하며 개발을 진행합니다. 논리적 사고 능력이 뛰어나 효율이 매우 높지만, 텍스트 기반 인터페이스가 낯선 초심자에게는 초기 진입장벽이 있을 수 있습니다

## 🏃‍♂️ 직접 해보자

### 환경 설정

**1. Node.js 설치**
1. [https://nodejs.org/ko/download](https://nodejs.org/ko/download) 접속
2. 하단 **Windows 설치 프로그램** 클릭 → 다운로드
3. 설치 마법사에서 계속 **다음** 클릭하여 설치 완료

**2. git 설치**
1. [https://git-scm.com](https://git-scm.com/) 접속
2. 우측 하단 **Install for Windows** 클릭 - 다운로드
3. 설치 마법사에서 계속 **다음** 클릭하여 설치 완료

**3. 작업 폴더 생성**
```
문서 > Vibe > study_01
```

**4. 폴더에서 터미널(프롬프트) 열기**

탐색기에서 `study_01` 폴더를 선택하고 우클릭 후 터미널에서 열기를 클릭합니다.
<img width="500" height="490" alt="image" src="https://github.com/user-attachments/assets/35318275-a21a-4746-b221-c34d048c01a6" />
```
cd Documents\Vibe\study_01
```

**5. 명령어 실행을 위한 터미널 설정 변경**

터미널에서 아래 명령어 복사해서 붙여넣고 엔터
```
Set-ExecutionPolicy RemoteSigned
```

**6. Claude Code 설치**

터미널에 아래 클로드 설치 명령어 복사 붙여넣고 엔터
```bash
npm install -g @anthropic-ai/claude-code
```

### 첫 실습


**1단계. Claude Code 실행**
```bash
claude 타이핑 후 엔터
```
1. 테마 선택
2. 클로드 인증 (Claude account with subscription 선택)
3. 옥타 인증 (회사 이메일)
4. 정상적으로 실행된 경우 아래와 같은 화면 표시
![Claude Code 실행 화면](https://github.com/user-attachments/assets/917746b5-165b-4472-8876-3b2b754f295a)


**2단계. 나만의 프로그램 만들기**
자유롭게 아이디어를 골라 터미널에 요청해보세요.
'
웹 계산기 만들어줘
파일명은 index.html이야
'

| 예시 아이디어 |
|---|
| 웹 계산기 |
| 맛집 추천 사이트 |
| 개인 프로필 사이트 |
| 오셀로 게임 |

**3단계. 디자인 입히기**

1. [https://wordpress.com](https://wordpress.com) 접속 → **테마** 메뉴 접속
2. 마음에 드는 테마 **캡처**
3. 캡처 이미지를 `study_01` 폴더 안에 저장
4. Claude Code에서 캡쳐 이미지를 선택하고 아래와 같이 요청
```
@이미지파일명 을 참고해서 디자인 변경해줘
```
> `@`를 입력하면 파일을 직접 선택할 수 있습니다.
>
