## 🎨 Claude Design으로 UI 만들어보기

**Claude Design**은 Anthropic이 새롭게 출시한 AI 디자인 도구예요.
채팅으로 설명하면 디자인·프로토타입·프레젠테이션을 바로 만들어줘요.
코딩 없이, 디자인 경험 없이도 OK!

> Pro / Max / Team / Enterprise 플랜에서 사용 가능 (연구 미리보기)

### Claude Design 작동 방식

| 영역 | 역할 |
|---|---|
| 왼쪽 채팅 | 원하는 것을 말로 설명하는 곳 |
| 오른쪽 캔버스 | Claude가 실시간으로 디자인을 그려주는 곳 |

기본 흐름
```
① 만들고 싶은 것 설명  →  ② 캔버스에서 결과 확인
③ 채팅 또는 인라인 댓글로 수정  →  ④ 내보내기 또는 공유
```

### 실습 순서

**1단계. Claude Design 접속**
1. [https://claude.ai](https://claude.ai) 접속
2. 좌측 메뉴에서 **Claude Design** 선택
3. **New Project** 클릭

**2단계. 디자인 요청하기**

아래 예시를 참고해서 자유롭게 요청해보세요.
```
월별 매출을 보여주는 대시보드 만들어줘. 지역별 필터도 있으면 좋겠어.
```
```
모바일 앱 온보딩 화면 4개 만들어줘. 핵심 기능을 소개하는 흐름이야.
```
```
고객 피드백 수집용 양식 만들어줘. 카테고리에 따라 질문이 달라져야 해.
```

**3단계. 수정 및 반복하기**

- **채팅** — 전체 레이아웃·색상 등 큰 변경에 사용
- **인라인 댓글** — 캔버스의 특정 부분을 클릭해서 세부 수정

```
색 구성표를 더 어둡고 미니멀하게 바꿔줘
```
```
이 버튼 패딩을 더 크게 만들어줘  (인라인 댓글)
```

**4단계. 내보내기**

오른쪽 상단 **내보내기** 버튼으로 다양한 형식으로 저장 가능

| 형식 | 용도 |
|---|---|
| PDF / PPTX | 발표 자료 |
| HTML (독립형) | 바로 배포 가능한 웹페이지 |
| Claude Code로 인계 | 실제 코드로 연결 |
| Canva로 전송 | 추가 편집 |

> 💡 완성된 디자인을 **Claude Code로 인계**하면 오늘 만들 DB 프로젝트와 바로 연결할 수 있어요!

## 🔌 MCP란 무엇인가

### 1. 개요

| 항목 | 내용 |
|---|---|
| **정식 명칭** | Model Context Protocol |
| **만든 곳** | Anthropic (Claude 개발사) |
| **발표 시점** | 2024년 11월 |
| **공개 방식** | 오픈소스 |
| **한 줄 정의** | AI가 외부 도구에 직접 접근하기 위한 공통 연결 규격 |

### 2. 만들어진 배경

MCP 이전에는 AI를 외부 서비스(GitHub, Notion, DB 등)에 연결하려면 **서비스마다 각각 다른 방식**으로 개발해야 했어요. 표준이 없어 같은 작업을 반복하는 비효율이 컸습니다.

Anthropic은 이를 해결하기 위해 'AI와 외부 도구의 연결 방식을 하나로 통일'하는 MCP를 만들었어요.

> USB-C가 다양한 기기의 충전 방식을 하나로 통일한 것처럼,
> MCP는 AI 연결의 공통 표준을 목표로 합니다.

| 시점 | 내용 |
|---|---|
| 2024년 11월 | Anthropic, MCP 공개 발표 (오픈소스) |
| 2025년 초 | OpenAI · Google 등 주요 AI 기업들도 MCP 지원 선언 |
| 2025년 현재 | 수천 개의 MCP 서버가 전 세계 개발자들에 의해 공개 중 |

### 3. MCP 전후 비교

**MCP 이전**
```
나: "내 GitHub 코드 봐줘"
Claude: "직접 접근이 안 돼요. 코드를 붙여넣어 주세요."
```

**MCP 이후**
```
나: "내 GitHub 코드 확인하고 버그 있으면 Notion에 정리해줘"
Claude: (GitHub 직접 접속 → Notion 직접 작성) "완료했어요!"
```

### 4. 연결 가능한 서비스 (예시)

| 분류 | 서비스 |
|---|---|
| 개발 | GitHub, Supabase |
| 업무 | Notion, Gmail, Slack, Google Calendar |
| 기타 | 웹 브라우저, 내 컴퓨터 파일 |

> 현재 수천 개 이상의 MCP 서버가 전 세계 개발자들에 의해 공개되어 있습니다.

### 5. 우리 스터디에서의 위치

```
지금  →  Supabase SDK 방식으로 기초 학습
다음  →  MCP로 Claude가 직접 접근하는 방식으로 심화
```

> 🚀 **"Claude에게 도구를 쥐어주는 것 — 그게 MCP입니다."**

## 🗄️ Supabase MCP 연결 — DB 프로젝트 만들기

Supabase는 데이터베이스(DB)를 쉽게 만들고 관리할 수 있는 서비스예요.
오늘은 MCP를 통해 Claude가 Supabase에 직접 접근해서 DB를 다루는 경험을 해봐요!

### Supabase란?

| 항목 | 내용 |
|---|---|
| 역할 | 데이터베이스 + 인증 + 스토리지를 한 번에 제공 |
| 비유 | 구글 스프레드시트처럼 데이터를 표로 저장하는 곳, 단 훨씬 강력함 |
| 왜 쓰나 | 백엔드 서버 없이도 데이터 저장·조회 가능 → 초보자에게 최적 |

### 실습 순서

**1단계. 작업 폴더 생성**
```
문서 > Vibe > study_03
```

**2단계. Supabase 프로젝트 생성**
1. [https://supabase.com](https://supabase.com) 접속 → 로그인 (GitHub 계정 권장)
2. **New Project** 클릭
3. 프로젝트 이름 입력 → DB 비밀번호 설정 → **Create new project**

> DB 비밀번호는 따로 메모해두세요! 나중에 환경변수 설정에 필요해요.

**3단계. Claude Code 실행 및 MCP 연결**

폴더에서 터미널 열고 Claude Code 실행
```bash
claude
```

Supabase MCP 연결 요청
```
supabase mcp 연결해줘
```

아래 두 가지 정보를 준비하세요.

| 필요한 정보 | 확인 위치 |
|---|---|
| **Access Token** | [https://supabase.com/dashboard/account/tokens](https://supabase.com/dashboard/account/tokens) → Generate new token |
| **프로젝트 URL** | Supabase 대시보드 → 내 프로젝트 선택 → **Settings → API → Project URL** |

입력한 정보는 Claude Code가 프로젝트 폴더 안의 **`.mcp.json`** 파일에 자동으로 저장해요.

> ⚠️ `.mcp.json`에는 Access Token이 저장돼요. `.gitignore`에 추가해서 GitHub에 올라가지 않도록 하세요!

**4단계. 아이디어 구상 및 요청**

DB가 필요한 서비스라면 뭐든 괜찮아요.

| 아이디어 | 저장할 데이터 |
|---|---|
| 팀 점심 메뉴 투표 | 메뉴 이름, 추천 수 |
| 사내 익명 건의함 | 건의 내용, 작성 시각 |
| 오늘의 한 줄 일기 | 날짜, 내용, 감정 태그 |
| 책 읽기 기록장 | 책 제목, 읽은 날짜, 한 줄 감상 |

Claude Code에 요청
```
supabase를 연결해서 [아이디어] 기능을 하는 웹페이지 만들어줘
데이터는 supabase에 저장되어야 해
```

**5단계. 결과 확인**

새 터미널을 열고
```bash
npm run dev
```

브라우저에서 접속
```
http://localhost:3000
```

> Supabase 대시보드 → **Table Editor**에서 내가 입력한 데이터가 실시간으로 쌓이는 걸 확인해보세요! 🎉
